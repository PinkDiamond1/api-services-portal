## General Usage

All Chakra `Input` props are valid props for the `TagInput` component.

<Canvas>
  <Story name="Default">
    <TagInput />
  </Story>
</Canvas>

The values should be passed as comma-separated strings.

```javascript
// Bad
<TagInput value={['Thing1', 'Thing2']} />

// Good
const value = JSON.stringify(['Thing1', 'Thing2'])
<TagInput value={value}/>
```

Though in most instances the API will be returning the stringified JSON array, and the component will handle the bookkeeping.

<Canvas>
  <Story name="Passing initial values">
    <TagInput value={JSON.stringify(['Thing1', 'Thing2'])}/>
  </Story>
</Canvas>

## Escape key clears the input value and blurs focus

Hitting escape when text is entered in the input will clear any non-tag values.
