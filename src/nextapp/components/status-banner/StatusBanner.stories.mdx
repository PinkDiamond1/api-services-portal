# Status Banner

The status banner component showcases status-related information to the user.

```javascript
import StatusBanner from '@src/components/StatusBanner';

const MyComponent = () => (
  <StatusBanner>
    <Heading>Status Banner Heading</Heading>
    <Text>Status Banner Description.</Text>
  </StatusBanner>
);
```

## Pending Status Variant

Note: when no `status` property is passed in, `pending` is used by default.
